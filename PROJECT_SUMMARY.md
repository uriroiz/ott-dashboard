# 📦 סיכום הפרויקט - דשבורד OTT

## 🎯 מטרת הפרויקט

דשבורד React מקצועי להצגת נתוני צפייה ב-OTT של איגוד הכדורסל הישראלי.  
כל חודש ניתן להעלות קובץ Excel עם נתוני הצפיות ולהציג דשבורד מפורט ללקוח.

## 📁 מבנה הפרויקט

```
ott-dashboard/
│
├── 📄 package.json              # תלויות והגדרות NPM
├── 📄 vite.config.js           # הגדרות Vite (Build Tool)
├── 📄 index.html               # HTML ראשי
├── 📄 .gitignore              # קבצים שלא נשמרים ב-Git
│
├── 📄 README.md                # תיעוד מלא
├── 📄 QUICKSTART.md           # התחלה מהירה
├── 📄 EXCEL_TEMPLATE.md       # מפרט קובץ Excel
├── 📄 PROJECT_SUMMARY.md      # המסמך הזה
│
├── 🔧 setup.bat                # סקריפט התקנה (Windows)
├── 🚀 start.bat                # סקריפט הפעלה (Windows)
│
└── src/                        # קוד המקור
    │
    ├── 📄 main.jsx            # Entry Point - נקודת כניסה
    ├── 📄 App.jsx             # קומפוננטת ראשית
    ├── 📄 index.css           # עיצוב גלובלי
    │
    ├── components/            # קומפוננטות React
    │   ├── FileUpload.jsx    # רכיב העלאת קבצים
```

## 📊 מה הדשבורד מציג?

### 1. תרשימים (3)
- **צפיות ומשתמשים לפי ליגה** - Bar Chart עם 2 datasets
- **שעות צפייה לפי ליגה** - Bar Chart
- **יחס צפיות לשעת הפקה** - Bar Chart

### 2. טבלאות (4)
1. **סיכום לפי ליגה** - נתונים מצטברים לכל ליגה (9 עמודות)
2. **סיכום בית** - 10 קבוצות מובילות במשחקי בית
3. **סיכום חוץ** - 10 קבוצות מובילות במשחקי חוץ
4. **סיכום כולל** - 10 קבוצות מובילות סך הכל

### 3. מטריקות
- מספר אירועים
- סך צפיות
- משתמשים ייחודיים
- שעות צפייה
- שעות הפקה
- יחס צפיות לשעת הפקה
- ממוצעים למשחק

## 🎨 עיצוב

- **כיוון:** RTL (מימין לשמאל) - מותאם לעברית
- **צבעים:** כחול (#1e40af) כצבע ראשי
- **Responsive:** מותאם לכל גודלי מסך
- **טיפוגרפיה:** Segoe UI
- **אנימציות:** חלקות ומקצועיות

## 🔄 תהליך העבודה

```
1. משתמש מעלה קובץ Excel
         ↓
2. xlsx קורא את הקובץ
         ↓
3. dataProcessor.js מעבד את הנתונים:
   - מזהה ליגות
   - מסנן לפי תאריך (עד 31/10/2025)
   - מסנן רק 6 ליגות רלוונטיות
         ↓
4. חישוב סטטיסטיקות:
   - סיכום לפי ליגה
   - סיכום בית/חוץ/כולל
         ↓
5. רינדור:
   - תרשימים (Charts.jsx)
   - טבלאות (LeagueTable.jsx, TeamTables.jsx)
   - הערות (Notes.jsx)
```
