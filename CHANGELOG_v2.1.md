#### 📋 טבלה כללית חדשה
**נוסף:** טבלה 2 - טבלה כללית חדשה

**תוכן:**
- כל הקבוצות מכל הליגות (או מליגה נבחרת)
- סך משחקים (בית + חוץ)
- סך צפיות, משתמשים, שעות צפייה
- ממוצעים למשחק

**מיקום:** בין טבלה 1 (סיכום ליגות) לטבלה 3 (בית)

**מטרה:** מבט מהיר על כל הקבוצות לפני צלילה לפירוט בית/חוץ

#### 📊 תרשימים חכמים
**שינוי התנהגות:**

**לפני:**
- תרשימים מוצגים תמיד (בכל הטאבים)

**אחרי:**
- **טאב "כל הליגות"** ← תרשימים מוצגים ✅
- **טאב ליגה ספציפית** ← תרשימים לא מוצגים ❌

**סיבה:**
- כשבוחרים ליגה ספציפית, אין טעם להשוות אותה לעצמה
- התרשימים מיועדים להשוואה בין ליגות
- בליגה בודדת - הטבלאות מספקות את כל המידע

#### דוגמה 1: מציאת קבוצה ספציפית

**לפני (v2.0):**
```

2. חפש בטבלה 2 (בית) - אולי היא בין ה-10 מובילות?
```

**אחרי (v2.1):**

### 💡 שיפורי UX

#### לפני
```
בוחר ליגה בודדת → רואה תרשימים של ליגה אחת בלבד (לא מועיל)
מחפש קבוצה → אולי היא לא ב-10 המובילות (תסכול)
```

#### אחרי
```
בוחר ליגה בודדת → רק טבלאות רלוונטיות (ממוקד)
מחפש קבוצה → תמיד נמצאת בטבלה 2! (נוח)
```
