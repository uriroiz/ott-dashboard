## 🔄 מיון טבלאות

### איך זה עובד?

```
1. לחץ על כותרת עמודה → מיון עולה ▲
2. לחץ שוב → מיון יורד ▼
3. לחץ פעם שלישית → חזרה למקור
```

### סימנים
- `⇅` = ניתן למיון (עדיין לא ממוין)
- `▲` = ממוין בסדר עולה
- `▼` = ממוין בסדר יורד

### דוגמאות מעשיות

#### מצא את הקבוצה הכי נצפית
```
1. לך לטבלה 2 (כללית)
2. לחץ על "צפיות (סה״כ)" פעמיים
3. הקבוצה הראשונה = הכי נצפית! ✅
```

#### מיין קבוצות לפי שם
```
1. לך לטבלה 2
2. לחץ על "קבוצה"
3. הטבלה ממוינת א-ת! ✅
```

#### מצא את הקבוצה עם הכי פחות משחקים
```
1. לך לטבלה 2
2. לחץ על "משחקים (סה״כ)" פעם אחת (עולה)
3. הקבוצה הראשונה = הכי מעט משחקים ✅
```

## 🏆 משחקים נצפים ביותר

### מה זה?
טבלה חדשה שמציגה את המשחקים הכי פופולריים!

### איפה זה?
**מיקום:** אחרי טבלה 1 (סיכום ליגות)

### כמה משחקים?
- **טאב "כל הליגות"** → 30 משחקים מכל הליגות
- **טאב ליגה בודדת** → 10 משחקים מהליגה הזו

### מה מוצג?
```
📅 תאריך
🏀 ליגה
🏠 קבוצת בית
✈️ קבוצת חוץ
👁️ צפיות
👥 משתמשים ייחודיים
⏱️ שעות צפייה
```

### דוגמה
```

```

```

```
